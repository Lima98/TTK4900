
For hundreds of years composers have been creating music using external tools like dice to help them generate something new. Creating a good solution for generating music procedurally has yet to be done and this thesis will explore a way to solve this problem.

\section{Context and Motivation}

Creating a good solution for generating good music will not just be applicable to generating music, but the structure will be able to be used for a number of other applications. That could be for other artistic purposes or any application where generating something that cannot be completely random.
